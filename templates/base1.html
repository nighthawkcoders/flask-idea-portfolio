About DNHS Computer Science

Collaboration

Interacting through code to converge towards a common goal.

Critical Thinking

Assessing why certain things might not work and coming up with a solution with a new approach.

Creativity

Creativity can be applied in creating new ideas for a code, algorithm, game, application and so on. Creativity helps generate new ideas, forming new projects that can implement your learning.

Communication

By communicating as a team, tasks and responsibilities can be performed effectively and efficiently. Not only can communication can be on different social platforms, but it can be seen through code as well by adding comments next to the code you write.
